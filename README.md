# 42.nibbler
 A snake game with multiple GUI library. The goal of this project is to learn how to use dynamic libraries. Therefore, it will use 3 differents dynamic libraries that the main executable will load and use at runtime.

## Objectives
Discover the power and usefullness of dynamic libraries used at runtime

Each dynamic library will embed everything the main executable might need to display the game and to get user's inputs.
The main executable will only focus on the game logic and interact with the dynamic libraries. The main executable must interact in a identical way with any of the libraries.
